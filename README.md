# Modern Live TV

A premium Live TV streaming web app built with Next.js 15, React 19, TypeScript, Tailwind CSS v4, shadcn-style primitives, Lucide icons, Framer Motion, HLS.js, TanStack Query, and Zustand.

## Features

- IPTV-org catalog integration for channels, streams, countries, languages, categories, and logos
- Home, Live TV, Watch, Favorites, History, Search, and Settings pages
- HLS live player with autoplay, retry, quality selector, fullscreen, volume, live status, keyboard shortcuts, and Picture in Picture
- Persisted favorites, watch history, search history, and playback settings
- Responsive desktop sidebar and mobile bottom navigation
- SEO metadata, sitemap, robots, JSON-LD, and PWA manifest

## Run

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Checks

```bash
npm run typecheck
npm run lint
npm run build
```
